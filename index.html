HELLO CSS

CSS semplifica drasticamente i farraginosi attributi, affibiati alla nostra pagina tramite HTML, selezionando specifici tag, e poi dando loro uno "stile". Letteralmente delle proprietà, dei parametri, attribuiti al singolo tag che abbiamo selezionato!
Diamo al Tag un identificatore univoco "ID", che può essere affibbiato ad uno ed uno solo elemento.
Altri elementi avranno bisogno di nuovi nuovi id
Più facile dar nome ad una classe per poterla attribuire ad uno o più elementi. "ID" è consigliabile usarlo esclusivamente per un solo elemento
